Постановка задачи. разработать программу, выполняющую следующие
функции:
Х ввод данных об n объектах из текстового файла в массив структур
(0<n<=50);
Х сортировку массива структур по возрастанию значений одного из полей
структуры;
Х вывод данных об объектах на экран в упорядоченном по возрастанию
виде;
Х поиск объекта по значению одного из полей;
Х запись упорядоченных данных об объектах в двоичный файл;
Х чтение двоичного файла.
Алгоритмы чтения файла, сортировки, поиска, вывода данных об объектах и
записи данных в файл оформить в виде функций. Для поиска элемента в
упорядоченном массиве использовать бинарный поиск. Текстовый файл
создать с помощью любого текстового редактора.
Варианты заданий приведены в табл. 11. В табл. 12 содержатся
описания функций классов библиотеки fstream для работы с файлами, в
табл. 18 Ц список режимов открытия файлов.
