Лабораторная работа 8
Вариант 4

Постановка задачи. Разработать программу, которая вводит матрицу из n
строк и m столбцов (n<=100, m<=50) и упорядочивает элементы матрицы.
Правило упорядочивания определяется вариантом. Программа должна
предоставлять пользователю выбор типа элементов матрицы: целого или
строкового. Операции ввода, вывода и упорядочивания элементов матрицы
должны быть реализованы в виде перегруженных функций (C++).

Упорядочить столбцы по убыванию первых элементов столбцов



*	Реализовать алгоритм:
	* Реализовать структуру Point.
	*	Реализовать матрицу двух типов:
		*	Для строк
		*	Для целых чисел
	*	Сортировка элементов матрицы (с помощью двух перегруженных функций):
		*	Для строк
		*	Для целых чисел
*	Реализовать интерфейс:
	*	Ввод целых чисел
	*	Ввод структуры x*y
	*	Реализовать вывод элементов матрицы (с помощью двух перегруженных функций):
		*	Для строк
		*	Для целых чисел
	*	Ввод матрицы (с помощью двух перегруженных функций):
		*	Для срок
		*	Для целых чисел


Конец.
